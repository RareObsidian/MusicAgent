```
## 第四步：乐理模块 - 音频播放

### 提示词
```
为钢琴助手添加音频播放功能，使用 Web Audio API 实现音阶播放。

要求：
1. 在乐理模块添加"播放音阶"按钮
2. 使用 Web Audio API 创建振荡器
3. 定义音符频率（C4=261.6Hz, D4=293.7Hz, E4=329.6Hz, F4=349.2Hz, G4=392.0Hz, A4=440Hz, B4=493.9Hz, #G4=830.6Hz）
4. 点击播放按钮时，按顺序播放当前调式的音阶
5. 每个音符播放 0.5 秒，间隔 0.1 秒
6. 使用三角波（triangle）模拟钢琴音色
7. 添加音量包络（attack/decay）使音色更自然
8. 播放过程中禁用按钮，防止重复点击
9. 播放时有视觉反馈（如按钮变色或显示动画）

音频实现参考：
```javascript
const ctx = new AudioContext();
const notes = {'C':261.6,'D':293.7,'E':329.6,'F':349.2,'G':392.0,'A':440,'B':493.9,'#G':830.6};

function playTone(freq, duration = 0.5) {
  const osc = ctx.createOscillator();
  const gain = ctx.createGain();
  osc.connect(gain);
  gain.connect(ctx.destination);
  osc.frequency.value = freq;
  osc.type = 'triangle';
  gain.gain.setValueAtTime(0.3, ctx.currentTime);
  gain.gain.exponentialRampToValueAtTime(0.01, ctx.currentTime + duration);
  osc.start();
  osc.stop(ctx.currentTime + duration);
}
```

输出：集成音频播放的乐理模块

计时器核心逻辑参考：
```javascript
let startTime, elapsed = 0, timerId;

function start() {
  startTime = Date.now() - elapsed;
  timerId = setInterval(() => {
    elapsed = Date.now() - startTime;
    display(formatTime(elapsed));
  }, 1000);
}

function pause() {
  clearInterval(timerId);
}

function formatTime(ms) {
  const totalSeconds = Math.floor(ms / 1000);
  const hours = Math.floor(totalSeconds / 3600);
  const minutes = Math.floor((totalSeconds % 3600) / 60);
  const seconds = totalSeconds % 60;
  return `${hours.toString().padStart(2,'0')}:${minutes.toString().padStart(2,'0')}:${seconds.toString().padStart(2,'0')}`;
}
```
