# 音乐学习应用 - 项目规划

## 项目概述
使用 Vue 3 + VexFlow 开发的音乐学习辅助应用，包含三大核心板块。

## 技术栈
- **前端框架**: Vue 3 (Composition API)
- **构建工具**: Vite
- **音乐符号渲染**: VexFlow
- **UI组件库**: Element Plus (可选)
- **数据可视化**: ECharts
- **状态管理**: Pinia
- **本地存储**: localStorage

## 项目结构
```
src/
├── components/           # 公共组件
│   ├── common/          # 通用组件
│   └── layout/          # 布局组件
├── views/               # 页面视图
│   ├── ScaleLearning/   # 板块1: 自然大小调音阶
│   ├── Pomodoro/        # 板块2: 番茄钟
│   └── Statistics/      # 板块3: 数据统计
├── stores/              # Pinia状态管理
│   ├── scaleStore.js    # 音阶学习状态
│   ├── pomodoroStore.js # 番茄钟状态
│   └── statsStore.js    # 数据统计状态
├── utils/               # 工具函数
│   ├── musicTheory.js   # 音乐理论相关
│   └── storage.js       # 本地存储封装
├── assets/              # 静态资源
└── App.vue              # 根组件
```

## 板块详细设计

### 板块1: 自然大小调音阶
**功能描述**: 使用VexFlow渲染五线谱，展示自然大调和小调音阶

**核心功能**:
- 选择调性(C大调、G大调、F大调等12个自然大调)
- 切换大小调模式
- 五线谱渲染显示音阶
- 音符播放功能(使用Web Audio API)
- 音阶练习模式(随机音符识别)

**页面布局**:
```
┌─────────────────────────────────────────┐
│  [调性选择]  [大调/小调切换]  [播放按钮]  │
├─────────────────────────────────────────┤
│                                         │
│         [五线谱显示区域 - VexFlow]        │
│                                         │
├─────────────────────────────────────────┤
│  [练习模式] [显示音名] [显示唱名]         │
└─────────────────────────────────────────┘
```

### 板块2: 番茄钟
**功能描述**: 专注练习计时器，帮助用户保持专注

**核心功能**:
- 可配置专注时长(默认25分钟)
- 可配置休息时长(默认5分钟)
- 倒计时显示
- 开始/暂停/重置控制
- 完成提醒(声音+通知)
- 练习记录保存

**页面布局**:
```
┌─────────────────────────────────────────┐
│           [圆形进度条计时器]              │
│              25:00                      │
├─────────────────────────────────────────┤
│      [开始]  [暂停]  [重置]             │
├─────────────────────────────────────────┤
│  [专注时长: 25分钟] [休息时长: 5分钟]     │
│  [今日专注次数: 3] [今日总时长: 75分钟]   │
└─────────────────────────────────────────┘
```

### 板块3: 数据统计
**功能描述**: 可视化展示学习数据和进度

**核心功能**:
- 练习时长统计(日/周/月视图)
- 音阶学习进度
- 番茄钟完成情况
- 连续学习天数
- 数据导出功能

**页面布局**:
```
┌─────────────────────────────────────────┐
│  [今日数据卡片] [本周数据卡片] [总数据卡片] │
├─────────────────────────────────────────┤
│                                         │
│      [练习时长趋势图 - ECharts折线图]      │
│                                         │
├─────────────────────────────────────────┤
│  [音阶掌握度 - 雷达图] [番茄钟统计 - 柱状图]│
├─────────────────────────────────────────┤
│  [连续学习天数: 7天] [导出数据按钮]        │
└─────────────────────────────────────────┘
```

## 数据模型

### 音阶数据
```javascript
{
  key: 'C',           // 调性
  mode: 'major',      // 模式: major/minor
  notes: ['C4', 'D4', 'E4', 'F4', 'G4', 'A4', 'B4', 'C5'],
  practiceCount: 10,  // 练习次数
  lastPracticed: Date // 最后练习时间
}
```

### 番茄钟记录
```javascript
{
  id: 'uuid',
  startTime: Date,
  endTime: Date,
  duration: 25,       // 实际专注时长(分钟)
  completed: true,    // 是否完成
  type: 'focus'       // focus/break
}
```

## 开发计划

### 第一阶段: 项目初始化
1. 创建Vue 3 + Vite项目
2. 安装依赖(VexFlow, ECharts, Pinia)
3. 配置项目结构和路由

### 第二阶段: 核心功能开发
1. 实现音阶学习板块
2. 实现番茄钟板块
3. 实现数据统计板块

### 第三阶段: 优化和完善
1. 添加动画效果
2. 响应式适配
3. 数据持久化
4. 测试和调试
