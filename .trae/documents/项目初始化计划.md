# 钢琴助手项目初始化计划

## 项目概述
- **项目名称**: MusicAgent（钢琴助手）
- **技术栈**: HTML + Tailwind CSS + 原生 JavaScript
- **目标**: 创建一个可运行的单页 Web 应用

## 核心功能
1. **乐理模块**: 3种调式查询（自然大调、自然小调、和声小调）
2. **练习计时**: 开始/暂停/结束 + 本地存储
3. **统计面板**: 今日时长 + 历史记录列表

## 初始化步骤

### 步骤 1: 创建基础文件结构
创建以下文件：
- `index.html` - 主页面（单页应用入口）
- `css/style.css` - 自定义样式（配合 Tailwind CDN）
- `js/app.js` - 应用逻辑

### 步骤 2: 编写 index.html
包含：
- HTML5 基础结构
- Tailwind CSS CDN 引入
- 页面布局（Tab 切换：乐理/练习/统计）
- 虚拟钢琴键盘 UI
- 计时器控制面板
- 统计面板

### 步骤 3: 编写 css/style.css
包含：
- 虚拟钢琴键盘样式（黑白键）
- 音阶高亮
