1. 产品概述

1.1 产品定位
钢琴助手是一款基于浏览器的钢琴学习辅助工具，无需安装，即开即用。集成乐理知识查询与练习数据追踪功能，支持桌面端和移动端浏览器访问。

1.2 目标用户
- 钢琴初学者（需要乐理知识支撑）
- 中级学习者（需要规律练习追踪）
- 音乐教师（用于学生练习监督）
- 任何有浏览器设备的用户

1.3 技术特性

特性	说明	
响应式设计	适配桌面（≥1024px）、平板（768px）、手机（<768px）	
PWA 支持	可安装为桌面/手机应用，支持离线使用	
本地优先	数据优先存储在浏览器本地，可选云端同步	
音频引擎	Web Audio API 实现实时音频合成

2. 功能模块

2.1 乐理知识模块

2.1.1 自然大调（Major Scale）

属性	说明	
音程结构	全-全-半-全-全-全-半（W-W-H-W-W-W-H）	
特征音	明亮、开阔的音响效果	
功能	查询任意调号的自然大调音阶组成	

示例：C自然大调

```
音级：  C   D   E   F   G   A   B   C
唱名：  do  re  mi  fa  sol la  ti  do
音程：  1   2   3   4   5   6   7   1
```

2.1.2 自然小调（Natural Minor）

属性	说明	
音程结构	全-半-全-全-半-全-全（W-H-W-W-H-W-W）	
特征音	柔和、忧郁的音响效果	
关系调	与自然大调共享相同调号（平行调）	

示例：A自然小调（C大调的关系小调）

```
音级：  A   B   C   D   E   F   G   A
唱名：  la  ti  do  re  mi  fa  sol la
```

2.1.3 和声小调（Harmonic Minor）

属性	说明	
音程结构	全-半-全-全-半-增二度-半	
特征音	升高第VII级音（导音），增强向主音的倾向性	
用途	和声进行、旋律写作	

与自然小调的区别

```
A自然小调：  A   B   C   D   E   F   G   A
A和声小调：  A   B   C   D   E   F   #G  A  ← 第VII级升高半音
```

2.1.4 交互功能
- 调号选择器：12个调号圆形布局（五度圈），支持点击/触摸选择
- 模式切换：自然大调 / 自然小调 / 和声小调 标签切换
- 音阶展示：
  - 虚拟钢琴键盘高亮显示（支持鼠标悬停/触摸）
  - 五线谱渲染（使用 VexFlow 或自研 Canvas 渲染）
  - 指法标注（可选显示）
- 音频播放：Web Audio API 合成播放，支持调节速度（60-120 BPM）

---

2.2 练习时长记录模块

2.2.1 核心功能

功能	描述	
计时器	精确到秒的练习计时，支持后台运行（Page Visibility API）	
练习分类	音阶练习、曲目练习、视奏练习、乐理学习、自由练习	
数据持久化	LocalStorage/IndexedDB 本地存储，数据不丢失	
统计视图	日/周/月/年 多维度图表展示（Chart.js）	

2.2.2 数据字段

```json
{
  "record_id": "uuid-v4",
  "start_time": "2026-03-28T14:30:00+08:00",
  "end_time": "2026-03-28T15:15:00+08:00",
  "duration_seconds": 2700,
  "practice_type": "曲目练习",
  "piece_name": "月光奏鸣曲第一乐章",
  "notes": "重点练习过渡段落",
  "focus_rating": 4,
  "created_at": "2026-03-28T15:15:05+08:00"
}
```

2.2.3 统计指标

指标	说明	可视化	
今日练习时长	当日累计分钟数	环形进度条	
本周练习时长	近7天累计	柱状图（周一至周日）	
连续练习天数	不间断练习 streak	火焰图标 + 数字	
平均单次时长	总时长 ÷ 练习次数	数字卡片	
练习类型分布	各类别占比	饼图/环形图	
月度趋势	近30天练习时长走势	折线图	

2.2.4 目标设定
- 支持设置每日/每周练习目标（默认：每日60分钟）
- 达成目标时：
  - 浏览器通知（Notification API）
  - 页面内动画庆祝效果
  - 成就徽章解锁

4. 技术架构

4.1 技术栈

层级	技术选型	
前端框架	React 18 / Vue 3（推荐 React + TypeScript）	
状态管理	Zustand / Pinia	
样式方案	Tailwind CSS + CSS Modules	
UI 组件	Headless UI / Radix UI（无障碍支持）	
图表库	Chart.js / Recharts	
音频引擎	Web Audio API + Tone.js	
五线谱渲染	VexFlow	
数据存储	IndexedDB（Dexie.js 封装）	
PWA	Workbox（离线缓存、Service Worker）	

4.2 数据存储策略

```
┌─────────────────────────────────────────┐
│           数据存储架构                   │
├─────────────────────────────────────────┤
│                                         │
│  用户操作  →  IndexedDB（本地优先）      │
│     ↓                                   │
│  网络可用?  →  同步至云端（可选）        │
│     ↓                                   │
│  多设备同步 ←  云端数据库（Firebase/自建） │
│                                         │
└─────────────────────────────────────────┘
```

本地存储结构：
- `settings`: 用户设置（主题、目标、偏好）
- `practice_records`: 练习记录（按日期索引）
- `achievements`: 成就徽章
- `cache`: 乐理数据离线缓存

4.3 音频实现

```javascript
// 使用 Web Audio API + Tone.js 示例
import * as Tone from 'tone';

// 初始化钢琴合成器
const piano = new Tone.PolySynth(Tone.Synth, {
  oscillator: { type: "triangle" },
  envelope: {
    attack: 0.005,
    decay: 0.1,
    sustain: 0.3,
    release: 1
  }
}).toDestination();

// 播放音阶
function playScale(notes, bpm = 90) {
  const now = Tone.now();
  notes.forEach((note, i) => {
    piano.triggerAttackRelease(note, "8n", now + i * (60/bpm));
  });
}
```

5. 功能规格

5.1 乐理模块详细规格

功能点	验收标准	
调号选择	点击调号后，键盘和五线谱实时更新，切换动画 < 300ms	
模式切换	三种调式切换时，特征音高亮显示（和声小调的#VII级用红色标注）	
虚拟键盘	支持鼠标悬停播放单音，点击音阶播放按钮播放完整音阶	
五线谱	正确显示调号、音符、升降号，支持导出为 PNG	
音频播放	支持 60-120 BPM 调节，播放时可暂停/停止	

5.2 练习模块详细规格

功能点	验收标准	
计时精度	误差 < 1秒，切换标签页/最小化后计时继续	
后台运行	使用 Page Visibility API + Web Worker 确保后台计时准确	
数据保存	练习结束自动保存，断网时本地存储，恢复网络后自动同步	
快捷操作	支持空格键开始/暂停，ESC 键结束练习	
通知提醒	练习目标达成时发送浏览器通知（需用户授权）
