# Vue VexFlow 音乐乐谱技能

此技能初始化一个 Vue 3 项目并集成 VexFlow 库，专门用于音乐符号渲染，特别是展示大调音阶的五线谱。

## 功能特点

- **Vue 3** 框架搭配 Vite 构建工具
- **VexFlow** 专业音乐符号渲染库
- **大调音阶展示** 支持所有 12 个调性
- **调号支持** 正确显示升降号
- **交互式调性选择** 下拉菜单
- **响应式设计** 适配不同屏幕尺寸

## 项目结构

```
project-name/
├── index.html              # HTML 入口
├── package.json            # 项目依赖
├── vite.config.js          # Vite 配置
└── src/
    ├── main.js             # Vue 应用入口
    ├── App.vue             # 主组件（包含调性选择）
    └── components/
        └── MusicScore.vue  # VexFlow 五线谱组件
```

## 项目依赖

- **vue**: ^3.5.31
- **vexflow**: ^4.2.5
- **@vitejs/plugin-vue**: ^5.2.4
- **vite**: ^5.2.0

## 使用方法

1. **初始化项目**：运行此技能创建一个新的 Vue 项目，自动配置 VexFlow
2. **选择调性**：使用下拉菜单选择任意大调音阶
3. **查看音阶**：在五线谱上查看带有正确升降号的音阶
4. **自定义扩展**：修改代码添加更多功能，如小调音阶、和弦进行或交互式练习

## 工作原理

- **App.vue**：提供用户界面，包含调性选择和音阶信息显示
- **MusicScore.vue**：使用 VexFlow 渲染五线谱，包括谱号、调号和音符
- **音阶计算**：为每个大调音阶计算正确的音符和升降号
- **八度处理**：确保音符显示在正确的八度范围内

## 示例输出

项目将显示一个网页，包含：
- 标题 "自然大调音阶展示"
- 调性选择下拉菜单（顺序：C, Db, D, Eb, E, F, Gb, G, Ab, A, Bb, B）
- 所选音阶的音符列表（例如："F - G - A - Bb - C - D - E"）
- 专业的五线谱，显示带有正确升降号的音阶
